## 🛠️ 快速开始

### 1. 系统初始化
```bash
# 查看系统信息
lsb_release -a
uname -a

# 检查用户权限
sudo whoami
groups
```

### 2. 执行基础配置
```bash
# 使用自动化脚本（推荐）
cd ../../scripts/setup/
./001-basic-setup.sh

# 或手动按照文档配置
cat 001-ubuntu-basic-system-standardized.md
```

### 3. 验证配置
```bash
# 运行验证脚本
cd ../../scripts/utils/
./system-verify.sh
```
